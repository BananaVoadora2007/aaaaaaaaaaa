Explorando Tags HTML e CSS Avançado

<h> - Títulos

A tag <h> é usada para definir títulos ou cabeçalhos em um documento HTML. Ela possui diferentes níveis, de h1 até h6, onde h1 é o mais importante e h6 o menos.

<p> - Parágrafos

A tag <p> é usada para definir parágrafos de texto em um documento HTML. Ela é utilizada para blocos de texto que formam parágrafos.

<img> - Imagens

A tag <img> é usada para incluir imagens em um documento HTML. Ela requer um atributo obrigatório src que especifica o caminho da imagem.
